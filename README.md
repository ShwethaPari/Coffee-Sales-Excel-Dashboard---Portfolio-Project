# Coffee Sales Excel Dashboard — Portfolio Project

## 📌 Overview
An end-to-end Excel project that transforms raw coffee order data into an interactive sales dashboard. The project covers data cleaning, lookups, calculated fields, PivotTables/PivotCharts, and interactive filtering with slicers and a timeline — built to demonstrate practical, job-ready Excel skills.

---

## 🛠️ Skills Demonstrated

* **XLOOKUP:** Pulled matching product and price information from a reference table.
* **INDEX & MATCH:** Implemented alternative lookup methods for flexible data referencing.
* **Advanced Formulas:** Utilized multiplication formulas for total sales (Price × Quantity) and nested `IF` functions for custom categorization.
* **Data Cleaning:** Handled date formatting, currency/number formatting, and duplicate checking.
* **Structured Data:** Converted raw data ranges into structured Excel Tables.
* **Data Visualization:** Built PivotTables and PivotCharts to summarize and visualize order data.
* **Interactive Filtering:** Implemented Slicers and Timelines for responsive dashboard filtering (by date range, category, etc.).
* **Dashboard Design:** Consolidated all visual elements into a single, cohesive, interactive dashboard sheet.
