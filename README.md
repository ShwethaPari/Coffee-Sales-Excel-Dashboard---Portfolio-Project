# Coffee Sales Excel Dashboard — Portfolio Project

**📌 Overview**
An end-to-end Excel project that transforms raw coffee order data into an interactive sales dashboard. The project covers data cleaning, lookups, calculated fields,
PivotTables/PivotCharts, and interactive filtering with slicers and a timeline — built to demonstrate practical, job-ready Excel skills.
️ 
**Skills Demonstrated**
**XLOOKUP** — pulling matching values (e.g.
product/price info) from a reference table
**INDEX MATCH** — alternative lookup method
for flexible referencing
**Formulas** — multiplication formulas for total
sales (Price × Quantity), multiple nested IF

functions for categorization
**Data cleaning** — date formatting,
number/currency formatting, duplicate
checking
**Tables** — converting raw ranges into
structured Excel Tables
**PivotTables & PivotCharts** — summarizing
and visualizing order data
**Slicers & Timeline** — interactive filters for the
dashboard (by date range, category, etc.)
**Dashboard design ** — combining all elements
into a single interactive dashboard sheet
